Professional migration tool to transfer content from Nuke PHP CMS to WordPress seamlessly and efficiently.

== Description ==

Nuke to WordPress Migration plugin provides a professional solution for migrating content from Nuke PHP CMS to WordPress. The plugin handles categories, articles, and images with batch processing capabilities and progress tracking.

**Key Features:**

* Migrate categories, articles, and images from Nuke to WordPress
* Batch processing to handle large datasets
* Progress tracking and resumable migrations
* Rollback capability for failed migrations
* Clean and modern admin interface using TailwindCSS
* Detailed logging and error reporting

**Requirements:**

* WordPress 5.8 or higher
* PHP 7.4 or higher
* MySQL/MariaDB database access to both Nuke and WordPress installations
* WordPress admin privileges

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/nuke-to-wordpress` directory, or install through WordPress plugins screen
2. Activate the plugin through 'Plugins' menu in WordPress
3. Navigate to 'Nuke to WP' in the admin menu
4. Configure your Nuke database credentials and migration settings

== Frequently Asked Questions ==

= Can I migrate specific content types only? =

Yes, you can select which content types (categories, articles, images) you want to migrate.

= What happens if migration fails? =

The plugin includes a rollback feature that allows you to revert any changes made during a failed migration.
